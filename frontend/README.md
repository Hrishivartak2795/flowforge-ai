# FlowForge Frontend

Next.js + React + TypeScript + Tailwind UI for FlowForge AI.

Server state is managed with TanStack Query (caching + status polling); the app talks
only to the backend REST API and holds no secrets client-side.

Scaffolded as a stub in **Milestone 0.6** and built out in **Milestone 10**.
See [`../docs/SystemDesign.md`](../docs/SystemDesign.md) §8 for the frontend architecture.
